# HANDOFF.md — session continuity

Write a handoff at the END of every working session (any agent, any vendor).
Newest entry on top. Keep each entry under ~30 lines — link, don't paste.

---

## Template

### YYYY-MM-DD — <agent/tool name> — Slice N
**Where I stopped:** one sentence.
**State of the gate:** `scripts/check.sh` result (ALL GREEN / failing sections).
**Done this session:** 2–5 bullets (commits by subject line).
**In flight / uncommitted:** anything staged, half-done, or intentionally left.
**Surprises captured:** kb ids added/updated, or "none".
**Next concrete step:** the exact first action for the next session.
**Watch out:** one thing the next agent would otherwise trip over.
